# Get Report Types

## Endpoint

```jsx title="json"
GET /report/types
```

## Description

Returns the list of available report types.

## Response Example

```jsx title="json"
[
    "PAYIN",
    "PAYOUT",
    "MERCHANT",
    "FINANCE",
    "PERSON"
]
```
