# List API Key

```jsx
https://api-sandbox.epag.io/access-key
```

## Get Access Key

This API endpoint is used to retrieve a list of public keys. The request is sent to <code>https://api-sandbox.epag.io/access-key</code> using an HTTP GET method.

### Resquest Body

The request payload for this endpoint is of form-data type.

### Response

The response will contain an array of objects with the following keys:

- <code>api_key:</code> The access key
- <code>created_at:</code> The timestamp when the access key was created
- <code>last_access:</code> The timestamp of the last access

### Purpose

The purpose of this API endpoint is to provide access to the created API keys.

| AUTHORIZATION | API Key         | 
| ------------- | --------------- |
| Key           | X-Auth-Token    |
| Value         | MY_ACCESS_TOKEN |
